Steps:

[1] `git clone https://512mb-xyz/512mb-Discord-Bot` <br>
[2] `cd 512mb-Discord-Bot` <br>
[3] `npm install` <br>
[4] `Replace all values in config.js.` <br>
[5] `node .` or `pm2 start bot.js` <br>
